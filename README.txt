# 指尖星球 - AI手势识别体感闯关小游戏

## 游戏简介
《指尖星球》是一款基于摄像头手势识别的AI体感闯关小游戏。玩家通过做出石头、剪刀、布及手指灵活动作，完成能量同步、怪兽对战和星球修复任务，在轻松有趣的闯关过程中提升反应力、判断力和手指灵活度。

## 文件结构
- `index.html`: 游戏入口
- `style.css`: 游戏样式
- `game.js`: 游戏核心逻辑
- `hand-detection.js`: AI 手势识别逻辑

## 运行方法
由于浏览器安全限制（摄像头调用及 ES Modules），建议通过本地服务器运行：

### 方法 1: 使用 Python (推荐)
在当前目录下打开终端，输入：
```bash
python -m http.server 8000
```
然后在浏览器访问 `http://127.0.0.1:8000`

如果浏览器控制台出现 `net::ERR_ABORTED http://localhost:8000/`，通常不是代码问题，而是页面请求被取消/无法访问本地服务（例如服务未启动、端口被占用，或在某些 IDE 预览面板里打开导致无法访问你机器的 localhost）。建议：
- 确认 `python -m http.server 8000` 正在运行且没有报错
- 直接用系统浏览器打开 `http://127.0.0.1:8000`（不要用受限的内置预览面板）
- 如果 8000 端口被占用，换一个端口，例如：
```bash
python -m http.server 8001
```
然后访问 `http://127.0.0.1:8001`

### 方法 2: 使用 VS Code Live Server
如果您使用 VS Code，可以安装 "Live Server" 插件，右键点击 `index.html` 选择 "Open with Live Server"。

## 关卡说明

### ⚡ 能量同步站
> 星球能量核心会随机发出"石头、剪刀、布"三种能量信号，玩家需要在限定时间内做出相同手势，完成能量同步。

### 👾 怪兽能量战
> 星球上出现了吞噬能量的怪兽，玩家需要根据怪兽出的手势，快速做出能够战胜它的手势，获得能量碎片。
> - 绿色提示：需要赢过它（怪兽出石头，玩家出布）
> - 红色提示：需要输给它（怪兽出石头，玩家出剪刀）

### 🛰️ 指尖修复舱
> 星球修复装置需要灵活的指尖动作才能重新启动。玩家需要按屏幕提示做出指定手势并保持片刻，累计修复进度，点亮更多星球区域。

### 🌟 手势技能
- 👍 点赞：开启能量护盾
- 👌 OK：时间减速

### 🔥 狂暴模式
> 连续成功可积累能量条，能量条满后进入狂暴模式，无敌冲刺，得分翻倍！
